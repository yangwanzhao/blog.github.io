Python中的卷积与fft

2020-09-03
  • convolution

在学习circulant矩阵与卷积的联系过程中，想用python验证相关公式，发现以往所学的卷积与fft的知识要么就是未求甚解，要么就是忘的一干二净，所以本文整理相关概念以及在python中的实现。由于只是在计算机上实现，所以对于fft只涉及离散的情况，不讨论繁杂的公式。

循环卷积与线性卷积

卷积分为循环卷积与线性卷积，因为只考虑离散情况下一维向量或二维矩阵的情况，所以只需要从向量角度理解这两种卷积如何操作，不需要考虑连续情况下的卷积公式。

若考虑向量h与向量x进行卷积：

循环卷积

循环卷积就是用h去加权x各个循环位移版本再求和

将__h__各点以x轴方向为起点逆时针排列成圆周，称为内圆；将__x__各点以x轴方向为起点顺时针排列成圆周，称为外圆。各位置元素对应相乘求和得到循环卷积的第一个结果。然后顺时针转动内圆各元素挪动一个位置，对应元素相乘求和，得循环卷积第二个结果。以此类推。

线性卷积

线性卷积就是用h去加权x各个时延版本再求和

将h以垂直方向镜像对称，将镜像后的最右边的元素与x的第一个元素对齐，各元素对应相乘，没有元素的位置补0，得到线性卷积的第一个结果。将镜像后的h往右挪动一个位置（现在h和x有两个位置重叠），各元素对应相乘，没有元素的位置补0，得到线性卷积的第二个结果。以此类推。

卷积定理

若$f_1(t)$经fft后为$F_1(w)$,$f_2(t)$经fft后为$F_2(w)$:

时域卷积定理

\[F[f_1(t)*f_2(t)]=F_1(w)\cdot F_2(w)\]

其中$*$表示卷积。该定理表明时域的卷积转换到频域后等于频域的乘积。

循环矩阵与fft

循环矩阵的定义: \(\mathrm{W}= \left[ \begin{matrix} w_1 & w_n & \cdots & w_3 & w_2 \\ w_2 & w_1 & w_n & & w_3 \\ \vdots & w_2 & w_1 & \ddots & \vdots \\ w_{n-1} & & \ddots & \ddots & w_n \\ w_n & w_{n-1} & \cdots & w_2 & w_1 \\ \end{matrix} \right]\)

即将第一列向量w逐次向下顺移得到其他列向量。

关于循环矩阵、fft以及卷积之间的重要公式： \(\mathrm{\textbf{W}}\mathrm{\textbf{x}}=\mathrm{ifft}(\mathrm{fft(\mathrm{w}) \circ \mathrm{fft}(\mathrm{x})})=\mathrm{w}*\mathrm{x}\) 其中$\circ$表示对应元素相乘，$*$表示卷积。

在该公式的帮助下，结合神经网络中$\mathrm{W}$大多情况为block-circulant，可以将神经网络的运算变换到频域进行，从而实现加速计算的目的。

Python函数

Python中的卷积函数

scipy.signal.convolve(X,W,mode='full')

适用于N-d情况

scipy.signal.convolve2d(X,W,mode='full')

当处理二维矩阵时，该函数有相同结果且更高效

以上两种计算卷积的方式均为计算__线性卷积__。

Python中的fft函数

numpy.fft.fft(x)

对信号做一维fft

numpy.fft.fft2(x)

等价于fft(fft(mat).T).T

值得一提的是，上述Python中卷积函数根据mode选项的不同，其结果维度也不同，当且仅当mode='same'的时候输出与输入同维度（在两个卷积项同维度的情况下）。然而，上述Python中的fft函数的输出总是与输入相同。这就导致当用卷积定理将二者联系起来的时候存在一些trick。从卷积定理的角度考虑，这里fft对应着循环卷积，因此在验证卷积定理的时候有两种情况：

  • 使用上述Python的卷积函数进行线性卷积，需要对被fft的信号进行padding。下文代码对信号进行尾部的补0，或将矩阵向右向下进行补0扩充。（[ ]为什么这样就能和线性卷积对应？）
  • 使用Python的卷积函数进行循环卷积，可直接fft并验证结果是否一致。下文代码中为手动计算循环卷积，如何用Python计算循环卷积有待完善。

代码实现

1d卷积定理

import numpy as np
from numpy.fft import *
from scipy import fftpack
from scipy import signal

X = np.array([2,1,3,4])
W = np.array([1,1,5,4])

print('** X **: ', X.shape)
print(X)

print('\n** W **: ', W.shape)
print(W)

grad_cir = np.array([25,35,30,20]) # calculate circular convolution manually
print('\n** Circular Convolution: ** \n  ** X # W **: ', grad_cir.shape) # '#' means convolve operation
print(grad_cir)

res1 = fft(grad_cir)
print('\n  ** res1 = f(X # W) **: ', res1.shape) # '#' means convolve operation
print(res1)

res2 = fft(X)*fft(W)
print('\n  ** res2 = F(X) * F(W) **: ', res2.shape)
print(res2)

# An N-dimensional array containing a subset of the discrete linear convolution
grad = signal.convolve(X,W,mode='full') 
print('\n** Linear Convolution: ** \n  ** X # W **: ', grad.shape) # '#' means convolve operation
print(grad)

res3 = fft(grad)
print('\n  ** res1 = f(X # W) **: ', res3.shape)
print(res3)

print('\n  ** pad X and W with N-1 0s to avoid circular convolution **')
X = np.append(X, [0]*(len(X)-1))
W = np.append(W, [0]*(len(W)-1))

res4 = fft(X)*fft(W)
print('  ** res2 = F(X) * F(W) **: ', res4.shape)
print(res4)

2d卷积定理

未涉及对二维矩阵的循环卷积

import numpy as np
from numpy.fft import *
from scipy import signal

X = np.random.rand(2,2)
W = np.random.rand(2,2)

print('** X **: ', X.shape)
print(X)

print('\n** W **: ', W.shape)
print(W)

# convolve is more suitable for N-d. 
# If you are processing 2-d, convolve2d has same results but is more efficient
# signal.convolve2d will reverse the W vertically and horizontally
# A 2-dimensional array containing a subset of the discrete linear convolution
grad = signal.convolve2d(X,W,mode='full') 

print('\n** grad=X # W **: ', grad.shape)
print(grad)

# fft2 <=> fft(fft(mat).T).T
res1 = fft2(grad)
print('\n** res1=F(X#W) **: ', res1.shape)
print(res1)

X = np.pad(X, ((0, X.shape[0]-1),(0,X.shape[1]-1)), 'constant', constant_values=(0,0))
W = np.pad(W, ((0, W.shape[0]-1),(0,W.shape[1]-1)), 'constant', constant_values=(0,0))

print('\n\nAfter Padding:')
print('** X **: ', X.shape)
print(X)

print('\n** W **: ', W.shape)
print(W)

res2 = fft2(X)*fft2(W)
print('\n** res2=F(X)*F(W) **: ', res2.shape)
print(res2)

circulant矩阵与fft

import numpy as np
from numpy.fft import *
from scipy import signal

W = np.array([[1,4,3,2],[2,1,4,3],[3,2,1,4],[4,3,2,1]])
X = np.array([6,3,5,1])
w = np.array([1,2,3,4])

print('\n** W **: ', W.shape)
print(W)

print('\n** X **: ', X.shape)
print(X)

print('\n** w **: ', w.shape)
print(w)

res1 = W.dot(X)
print('\n** res1=W.X **: ', res1.shape)
print(res1)

res2 = ifft(fft(w)*fft(X))
print('\n** res2=iF(F(w)*F(X)) **: ', res2.shape)
print(res2)

# Linear Convolution
res3 = signal.convolve(w,X,mode='full') 
print('\n** res3=w#X Linear**: ', res3.shape)
print(res3)

# Circular Convolution
res4 = np.array([35,38,33,44])
print('\n** res4=w#X Circular**: ', res4.shape)
print(res4)
